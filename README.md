# SVsim
섀도우버스 게임 엔진 (개발 중)
이 프로젝트는 카드 게임 섀도우버스 월즈 비욘드의 핵심 게임플레이를 처리하는 엔진을 구현하는 것을 목표로 합니다. 현재 적극적으로 개발 중이며, 게임의 복잡한 규칙과 상호작용을 처리하기 위한 견고하고 확장 가능한 기반을 구축하고 있습니다.

## 1. 현재 개발 현황
현재까지 다음과 같은 주요 시스템 및 규칙의 초기 구현이 진행되었습니다:

 - 카드 카테고리: 추종자, 주문, 마법진 카드 타입 정의 및 기본 동작.
 - 핵심 키워드 능력: 카운트다운, 교전시, 공격시, 질주, 수호, 진화시, 잠복, 돌진, 흡혈, 유언, 위압, 오라, 배리어 키워드 능력의 발동 로직 일부 구현.
 - 게임 진행: 초기화(덱 구성 (40장, 3장 제한), 초기 4장 드로우 및 멀리건 단계), 턴 진행
  - PP (플레이 포인트) 시스템: 턴별 PP 증가 및 회복, 카드 사용 시 PP 소모 로직.
 - 게임 영역 관리: 덱, 패, 전장, 묘지 간의 카드 이동 및 관련 규칙 (예: 패 제한 9장, 전장 제한 5장).
