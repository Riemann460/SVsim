# SVsim
섀도우버스 게임 엔진 (개발 중)

이 프로젝트는 카드 게임 섀도우버스 월즈 비욘드의 핵심 게임플레이를 처리하는 엔진을 구현하는 것을 목표로 합니다. 현재 적극적으로 개발 중이며, 게임의 규칙과 상호작용을 처리하면서 확장 가능한 기반을 구축하고 있습니다.

## 1. 개발 진행 현황
현재까지 다음과 같은 주요 시스템 및 규칙의 초기 구현이 진행되었습니다:

 - 게임 영역 관리: 덱, 패, 전장, 묘지 간의 카드 이동 및 관련 규칙
 - 게임 진행
   - 초기화(덱 구성 (40장, 3장 제한), 초기 4장 드로우 및 멀리건)
   - 턴 진행(PP 변화, 드로우, 공격 상태 초기화)
   - 카드 플레이 및 추종자 조작(공격/진화/초진화)
   - 카드 효과 처리(대상 지정 능력은 임시로 무작위 대상으로 처리 중)
 - 키워드 능력: 교전시, 공격시, 수호, 흡혈, 유언, 위압, 오라, 배리어, 증강, 질주, 돌진, 잠복, 배리어, 진화시, 초진화시, 필살, 카운트다운
 - 카드 : 중립 카드
